# 산단 교대시간대 R5 시뮬레이션

R5 실제 출력: `sandan_accessibility_0600_by_stop.csv`, `sandan_accessibility_0800_by_stop.csv`, `sandan_accessibility_2200_by_stop.csv`.

집계 기준: 고용>0 산단 20개, 산단별 정류장 수요가중 접근분을 다시 계산한 뒤 고용가중 평균 산출.

| 출발 | 고용가중 평균 접근분 | 60분내 도달 산단 | 도달불가 산단 |
|---|---:|---:|---:|
| 06:00 | 90.2분 | 0/20 | 0 |
| 08:00 | 86.8분 | 0/20 | 0 |
| 22:00 | 80.4분 | 0/20 | 0 |

- 06:00는 08:00 대비 +3.4분.

- 22:00는 08:00 대비 -6.4분.

도달불가 산단은 해당 산단 컬럼에서 수요>0 출발정류장 전체가 R5 travel_time 결측인 경우로 계산했다.
